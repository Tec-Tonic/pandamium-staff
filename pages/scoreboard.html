How to use

Please make sure that only ONE players scoreboard data is in your latest.log file! If more than one players data is present in your log file please copy the relevant data and place that in a .txt file to be uploaded
- Navigate to your latest.log file. More info Below
- Make sure the Old Username is correct once you upload the latest.log file!

Log Folder:
Windows: Type %appdata% in the address bar of the Windows Explorer. Then open the .minecraft folder.
Mac: Press Cmd+Shift+G then type: ~/Library/Application Support/minecraft in the search box.

In your logs folder there will be a file called latest.log use that one.
